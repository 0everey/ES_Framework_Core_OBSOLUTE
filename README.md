# 第二部分 术语表
[^ES]: ES就是本框架的总命名空间，很多功能完成后类型接口名都可能以ES开头来做区分防止和其他内容冲突
[^托管器]: 托管器，原始接口为 IESHosting 就是能容纳模块，并且控制它的生命周期的类，而且托管器也可能是模块
[^模块] : 模块 ,原始接口为 IModule ,他是一系列可序列化类，可以执行功能并且具有严格
